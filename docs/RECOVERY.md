# 恢复流程

1. 克隆此公开模板；
2. 安装依赖；
3. 将 `.env.example` 复制为 `private-overlay/.env`；
4. 仅在本地填写真实值，不要提交私有覆盖层；
5. 把模板安装到独立运行目录；
6. 应用私有覆盖层；
7. 根据 `sql/schema.sql` 初始化新的空数据库；
8. 使用新的 ID 导入已经复核的自动化示例；
9. 启动 OpenClaw Gateway；
10. 运行测试和构建，并手动验证高风险确认流程。

公开模板无法恢复以下个人数据：

```text
Token
地址
账号
个人记忆
历史运行状态
```

这些内容必须来自仓库所有者单独保存的加密私有备份。
